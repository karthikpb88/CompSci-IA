This project was written in python 3.7 as specified by the client. While python is an interpreted language that should theoretically work across multiple platforms, this project makes use of special system calls through the Python OS library specific to MacOS/Linux (POSIX) systems. 


It also makes use of the UNIX Filesystem Hierarchy Standard, and therefore may not run correctly on windows environments. 


However, if you would like to view the source code as is, please navigate to the Product folder and view the source in a text editor that supports Python syntax highlighting. To view the product filesystem, navigate to Product/FileSystem and explore the various directories and files that power this project.


To run the source code and interact with the project, follow the pertinent steps for your respective operating system:



  1. MacOS (Unix/Linux):- Click on the Link to: Product hyperlink given on the HTML cover sheet to launch the Python_code directory on your desktop. Find the the “Product(UNIX).command” file and double click it to launch the program. Alternatively, navigate to Python_code within Product folder of the IA using the system terminal. Execute the included bash script by entering  "sh ./product.sh" to run the product on your local environment.
  2. Windows:- This product is currently untested on windows environments, and may or may not work with limited functionality. Make sure that a python 3.x.x environment is installed and correctly set up on your local computer, and system environment variables are pointed to python. Navigate to the Python_code directory under Product/ folder. Find the “Product(win).bat” file and double click to launch the program. Alternatively, simply drag and drop the "Interface.py" script from the Python_code folder within Product/Python_code onto a cmd terminal window and press enter to launch the product.
